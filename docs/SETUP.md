# Setup Guide

> **Note**: Detailed setup instructions will be added in Phase 11.

## Prerequisites

- Python 3.10+
- Node.js 18+
- Ollama with Phi-3 Mini model

## Quick Start

```bash
# Backend
cd backend
poetry install
poetry run uvicorn app.main:app --reload

# Frontend
cd frontend
npm install
npm run dev
```
